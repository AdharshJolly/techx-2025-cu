# TECHX 2025

Flagship IEEE Technical & Innovation Event hosted by IEEE Computer Society - CHRIST University.

## Overview
3 days of AI-driven workshops, prompt engineering battles, and the ultimate VIBEATHON innovation sprint.

## Features
- **Sponsors Marquee**: Dynamic scrolling partners.
- **Journey Timeline**: Interactive 3-day roadmap.
- **Workshops Dashboard**: Dual-track learning and competition.
- **Prizes Podium**: Winner recognition and pool distribution.
- **Holographic Tickets**: Secure access passes.
